Backend (qiskit_dynamics.backend)Â¶

This module contains the DynamicsBackend class, which provides a BackendV2 interface for running pulse-level simulations with Qiskit Dynamics. The DynamicsBackend can directly simulate Schedule and ScheduleBlock instances, and can also be configured to simulate QuantumCircuits at the pulse-level via circuit to pulse transpilation.

This module also exposes some functions utilized by DynamicsBackend that may be of use to experienced users. The function default_experiment_result_function() is the default method by which results are computed and returned to the user after the underlying differential equation is solved. This function can be overridden with a custom user-defined function by setting the experiment_result_function option of DynamicsBackend. The parse_backend_hamiltonian_dict() function is used by DynamicsBackend.from_backend() to construct model matrices from the backend Hamiltonian description. The function documentation gives a detailed explanation on the expected input formatting.
